自分の気持ちを、短く、はっきり伝える

いまの気持ちカード

自閉スペクトラム症の人が、いまの感情、しんどさ、困りごと、してほしい支援を、 選択肢と自由入力で伝えるためのシンプルなページです。

すぐに使う

よくある状況を押すと、自動で入力されます。

1. いまの気持ち

1つ選んでも、複数選んでも大丈夫です。

2. しんどさの強さ

0は落ち着いている、5はかなりつらい状態です。

3. いま困っていること

体や環境のつらさも選べます。

4. してほしい支援

相手にしてほしいことを選んでください。

5. 自由に書く

6. 表示する文章

下の文章は自動で作られます。必要なら自由に直してください。

使い方

  1. 当てはまる気持ちや困りごとを押します。
  2. 必要なら自由入力を足します。
  3. 「文章を作る」を押して、そのまま見せるかコピーします。

このページの考え方

このページは、視覚的な選択肢、AAC、本人の好みや支援ニーズの明示、 そしてOSのダークモード設定を尊重する実装を前提に設計しています。

詳しい参考文献は README にまとめています。
